# Animation Packs

Ready-to-use SLSB conversions for popular animation packs.

---

## About SLSB Conversions

SexLab P+ uses **SLSB** (SexLab Scene Builder) instead of SLAL for animation registration. SLSB is an external tool that compiles animation definitions into `.slr` (SexLab Registry) files (along with modified FNIS AnimLists) that P+ loads at startup.

Most existing SLAL packs need conversion patches to work with P+.

### How to Install

1. Download the **original SLAL pack** (for meshes and assets)
2. Download the **SLSB conversion** (contains compiled `.slr` files as well as modified FNIS AnimLists and behaviour files)
3. Install both, letting the SLSB conversion overwrite

> **Note:** Until authors grant redistribution permission, both the original pack and conversion must be downloaded separately.

### The Automated.SLSB.Conversions Package

The community maintains **Automated.SLSB.Conversions**, a bundled package containing pre-compiled SLSB conversion patches for many popular SLAL packs. This allows you to download multiple conversions in a single file instead of downloading each one individually.

**How it works:**
- Download the Automated.SLSB.Conversions package from the Discord
- Install it after your SLAL animation packs, letting it overwrite them
- The conversions will apply automatically

This is especially useful if you're installing many animation packs at once.

### Want to Learn More?

- See [SLAL vs SLSB](/slsb/slal-vs-slsb/) for a technical comparison
